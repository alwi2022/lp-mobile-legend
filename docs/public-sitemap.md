# Public Sitemap

Struktur halaman publik saat ini:

- `/`
  - Hero utama
  - CTA daftar ke `/register`
  - Ringkasan turnamen
  - 3 pertandingan terdekat
  - Featured stream
  - Preview bracket
  - Preview tim

- `/jadwal`
  - Jadwal lengkap
  - Filter event / pekan
  - Filter status pertandingan

- `/bracket`
  - Bracket penuh
  - Daftar match per ronde

- `/tim`
  - Daftar seluruh tim resmi

- `/siaran`
  - Featured stream
  - Daftar seluruh siaran
  - Pilih stream yang ingin ditampilkan

- `/faq`
  - Tanya jawab umum turnamen
  - Konten masih dummy

- `/register`
  - Form pendaftaran publik

Prinsip halaman:

- Homepage dipakai untuk ringkasan dan navigasi cepat.
- Data yang panjang atau sering dibuka dipindah ke halaman khusus.
- Struktur ini dibuat supaya website lebih mirip pola turnamen profesional:
  - home untuk highlight
  - page terpisah untuk data operasional
